사용한 이벤트 목록

사용한 이벤트 5가지 이상 정리

1. submit 이벤트

onsubmit은 폼이 제출될 때 실행.

2. click 이벤트

click은 버튼 등을 클릭할 때 발생. 여기선 submit 버튼 클릭 시 onsubmit이 작동.

3. change 이벤트

onchange는 입력값이 변경되었을 때 작동. 생년월일을 바꾸면 나이를 자동 계산하는 데 사용.

4. DOMContentLoaded 이벤트

문서의 모든 요소가 로드되면 실행. 회원가입창에서 초기 나이 계산을 위해 사용.

5. alert 함수

alert()는 이벤트 결과로 메시지를 사용자에게 보여줄 때 사용.

6. checkbox, radio 관련 이벤트

change는 체크박스나 라디오 버튼의 값이 바뀔 때 사용.

7. load 이벤트

load는 각 이미지가 브라우저에 로드될 때 발생.

8. mouseover 이벤트

mouseover는 사용자가 이미지나 링크 위에 마우스를 올리면 발생.

이 외에도 여러 이벤트가 사용되었지만, 공간의 효율을 위해 이만 줄입니다.
